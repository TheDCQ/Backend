The Daily Coding Question

Sum of Odd Integers

Statement

You are given two integers n and k. Find if n can be written as the sum of k distinct positive odd integers.

Input

A single line, n and k, separated by a single space.

Output

For each test case, print k distinct positive odd integers which have the sum n. If that is impossible, print -1 .

Example

Input

16 4

Output

1 3 5 7

Input

10 3

Output

-1
